# 🎯 Class 10 Syllabus Tracker

A high-performance web application designed to help Class 10 students track their academic progress with precision and style.

### ✨ Key Features
* **Modern Dashboard:** Attractive circular progress rings with rounded caps.
* **Subject-Wise Tracking:** Detailed breakdown for Maths, Science, SST, English, and Hindi.
* **Smart Notifications:** Automated milestone alerts (every 10%) and a personal notice board.
* **PWA Enabled:** Install it on your phone or PC and use it **offline**.
* **Data Privacy:** All progress is saved locally on your device using `localStorage`.

### 🛠️ Built With
* **HTML5 & CSS3:** For the sleek, wavy-header design and responsive layout.
* **JavaScript (ES6):** For real-time progress logic and SVG animations.
* **Progressive Web App (PWA):** Service Workers for offline reliability.
